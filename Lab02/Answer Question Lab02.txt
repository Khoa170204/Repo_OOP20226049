+ Question 1: When should accessor methods be used? (“Why getter and setter methods are evil” JavaWorld, 5 Sep. 2003)
=> Answer:
Provide controlled access to the private instance variables of a class
Prevent other classes from accessing and modifying them directly.
Accessor methods provide a way for other classes to access and modify the private instance variables in a controlled manner.

+ Question 2: If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this? 
=> Answer: No , because they have the same signatures.
